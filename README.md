# OpenCamBridge

OpenCamBridge is a free open-source phone-as-webcam project.

V1 target:
- Android phone camera over Wi-Fi
- Android USB via adb reverse
- Browser MJPEG stream
- Local HTTP control API
- No cloud
- No account
- No ads
- No telemetry
- No watermark

Not in V1:
- iOS implementation
- Bluetooth video
- WebRTC
- Windows/macOS virtual camera driver
- HEVC
